#### 常量

ES5常量写法：

```js
Object.defineProperty(window, "PI2", {
	value: 3.1415926,
	writable: false,
})
console.log(window.PI2);
```

ES6常量写法

```js
const PI = 3.1415926;
console.log(PI);
```
